A Web-Development Journey

After watching some YouTube Videos, I thought to my self: It has never been so easy for me to learn coding. On the one hand, regarding my skill-level and general understanding of how programming works. On the other hand regarding the infinite number of high quality learning material you can access for free.

First Steps

I started out learning some basic Python algorithms on Python for Everybody. Nothing too new for me, because most of the content was part of my Programming 1 and Programming 2 course during my studies.

Preparation

While learning Python became a little bit boring, I started working myself through "The Odin Project". The first lessons were extremely interesting and I truly understood what the Internet is and the abstract concept of "the Internet" became a logical one to me.

What's happening now

Right now I'm practising some HTML basics. While I practise, I'm curious about how to put Websites on a server so everyone can visit them. I will continue with learning more HTML stuff and then head over to CSS and JavaScript. I can't await it.
